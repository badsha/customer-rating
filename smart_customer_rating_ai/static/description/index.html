Each customer is automatically categorized using intelligent badges.

  • High Value — consistent purchases and strong payment history
  • VIP — top-tier engagement and revenue contribution
  • Payment Risk — delayed or irregular payments detected
  • Low Engagement — declining or minimal activity

Local AI with Ollama (Optional)

Prefer keeping AI on your own server? Choose AI Provider: Local Ollama in Settings and run an Ollama instance locally (no external API key required).

  • Default URL: http://localhost:11434
  • Default Model: llama3
  • If Odoo runs in Docker, set the URL to your host (for example: http://host.docker.internal:11434)

Docker quick start: 

docker run -d --name ollama -p 11434:11434 -v ollama:/root/.ollama ollama/ollama
docker exec -it ollama ollama pull llama3
